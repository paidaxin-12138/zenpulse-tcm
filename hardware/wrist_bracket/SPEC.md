# 腕带托板机械规格（3D 打印）

> 适用于双 MAX30102 桡动脉腕带原型 · 单位 mm

## 1. 总体尺寸

| 参数 | 值 |
|------|-----|
| 外形（长 × 宽 × 厚） | 52 × 34 × 9 |
| 底面弧半径 R | 40（贴合腕背） |
| 材料 | PLA / PETG |
| 打印方向 | 底面弧面朝下（需支撑）或分体打印 |

## 2. 传感器安装位

两颗 MAX30102 模块（常见 14×14 mm PCB）：

| 传感器 | 中心距托板左缘 | 中心距底弧最低点 | 备注 |
|--------|----------------|------------------|------|
| #1 关/寸 | 14 | 0（弧底） | 主通道 CH1 |
| #2 冗余 | 26 | 0 | 近心侧 CH2，间距 12 mm |

- 安装孔：M2，孔距按模块丝印  
- LED 窗口开孔：Ø5.5，居中于模块 LED  
- 窗口周围 **2 mm 遮光裙边** 高度 1.5 mm（贴 O 圈）

## 3. 腕带槽

- 两侧各一条通槽：宽 22 mm（适配 20 mm 魔术贴）  
- 槽距中心 48 mm  
- 传感器区域 **不在** 魔术贴覆盖范围内

## 4. ESP32 安装

- 顶面 4×M3 孔，矩形 **52×28 mm**（ESP32-S3 DevKitC-1）或 **50×26 mm**（经典 ESP32）
- 或用 3D 打印小盒置于腕带外侧，I2C 线长 ≤ 15 cm

## 5. OLED 显示屏（外侧主控仓）

| 参数 | 建议 |
|------|------|
| 型号 | 0.96" SSD1306，128×64，**I2C**（4pin：VCC/GND/SDA/SCL） |
| 地址 | 0x3C（少数为 0x3D） |
| 安装 | 主控盒 **顶面开窗**，屏幕用双面胶或 M2 柱固定 |
| 固件 | `config.h` → `ENABLE_OLED 1` |
| 显示内容 | BLE 状态、信号 OK、本地 HR 估算、IR 读数 |

**手环集成布局**

```
[ 腕背外侧 ]                     [ 桡侧贴皮肤 ]
┌─────────────────────┐         ┌──────────────┐
│ OLED (128x64)       │         │ MAX30102 ×2  │
│ ESP32-S3 + 电池      │◄─I2C───►│ 遮光 O 圈     │
└─────────────────────┘         └──────────────┘
        魔术贴腕带环绕
```

## 7. OpenSCAD 参数化（可选）

见同目录 `bracket.scad`（若存在）。无打印机时可暂用：

- 双层 Eva 板手工开孔  
- 或 3D 打印店提供 STL，导出参数：长 52 宽 34 高 9，R40 弧底
