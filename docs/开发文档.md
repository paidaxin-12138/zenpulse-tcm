## 1. 项目概述

### 1.1 项目背景
中医智能诊断系统是基于现代人工智能技术与传统中医理论相结合的创新型医疗辅助工具。系统通过分析患者的面部特征、舌苔情况、眼部特征以及生理指标，结合海量中医知识和临床案例，为用户提供个性化的中医诊断和调理建议。

### 1.2 项目目标
- 构建一个包含十几万字权威中医资料的知识库
- 收集和管理5000例真实临床案例
- 实现基于计算机视觉的中医四诊（望诊）自动化
- 提供智能中医诊断和个性化调理建议
- 支持Web端访问，方便用户使用

### 1.3 核心价值
- 传承和推广中医文化
- 提高中医诊断的效率和准确性
- 为用户提供便捷的中医健康咨询服务
- 为中医研究提供数据支持

## 2. 技术架构

### 2.1 技术栈
- **后端框架**: FastAPI

- **前端**: HTML, CSS, JavaScript
- **开发语言**: Python 3.10+

## 4. 核心功能模块

### 4.1 中医知识库管理
- **功能描述**: 管理和加载中医知识，包括基础理论、临床案例、中药学等
- **实现方式**: 使用文件系统存储知识，通过Chroma向量数据库进行索引和检索
- **关键特性**: 支持海量中医资料，提供高效的语义搜索功能

### 4.2 计算机视觉分析
- **功能描述**: 分析患者的面部、舌苔和眼部特征
- **实现方式**: 使用OpenCV进行图像分析，结合颜色空间转换和特征提取算法
- **关键特性**: 自动化分析，提高诊断效率

### 4.3 智能诊断模块
- **功能描述**: 根据视觉分析结果和生理指标，提供中医诊断和调理建议
- **实现方式**: 集成LLM模型，结合中医知识库进行智能分析
- **关键特性**: 个性化诊断，基于权威中医知识

### 4.4 Web服务
- **功能描述**: 提供Web访问界面，支持用户上传图片和查看诊断结果
- **实现方式**: 使用FastAPI框架构建RESTful API
- **关键特性**: 高性能，易于扩展

## 6. 部署与运行指南

### 6.1 环境要求
- Python 3.10+
- Windows 10+
- 至少8GB RAM
- 至少10GB可用磁盘空间

### 6.2 依赖安装
```bash
pip install -r requirements.txt
```

### 6.3 启动步骤
1. 确保所有依赖已安装
2. 运行Web服务器
   ```bash
   python web_server.py
   ```
3. 打开浏览器，访问http://localhost:8000

### 6.4 常见问题解决
- **端口被占用**: 修改web_server.py中的端口号
- **向量数据库初始化失败**: 确保vector_store目录有写入权限
- **模型加载失败**: 确保models目录下有完整的模型文件

## 7. API接口文档

### 7.1 健康指标接口

#### 7.1.1 获取随机健康指标
- **URL**: /api/health-metrics
- **方法**: GET
- **标签**: 健康指标
- **描述**: 生成随机健康指标用于测试
- **响应示例**:
  ```json
  {
    "heart_rate": 75,
    "pulse": 75,
    "systolic": 120,
    "diastolic": 80
  }
  ```

### 7.2 诊断接口

#### 7.2.1 中医智能诊断
- **URL**: /api/diagnose
- **方法**: POST
- **标签**: 诊断
- **描述**: 根据上传的图片和健康指标进行中医诊断
- **参数**:
  | 参数名 | 类型 | 描述 | 是否必填 |
  |--------|------|------|----------|
  | tongue_image | File | 舌苔照片 | 否 |
  | face_image | File | 面部照片 | 否 |
  | eye_image | File | 眼睛照片 | 否 |
  | heart_rate | int | 心率 | 是 |
  | pulse | int | 脉搏 | 是 |
  | systolic | int | 收缩压 | 是 |
  | diastolic | int | 舒张压 | 是 |

- **请求示例**:
  ```http
  POST /api/diagnose
  Content-Type: multipart/form-data
  
  heart_rate=75
  pulse=75
  systolic=120
  diastolic=80
  tongue_image=@tongue.jpg
  face_image=@face.jpg
  ```

- **响应示例**:
  ```json
  {
    "diagnosis": "1. 中医辨证：气血不足\n2. 证候分析：根据望诊和健康指标分析，您的身体状况表现为气血不足。\n3. 建议：1. 饮食均衡，多食用补气血的食物；2. 保持充足睡眠；3. 适当运动；4. 定期体检。",
    "source": "中医智能诊断系统(LLM模式)"
  }
  ```
